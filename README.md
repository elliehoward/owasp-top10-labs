\# OWASP Top 10 Practice



This repository is actively being updated as I complete new OWASP labs.



This repository documents my hands-on practice with OWASP Top 10 vulnerabilities using platforms like TryHackMe and PortSwigger.





\## Topics Covered



* Authentication \& Authorization Failures
* Insecure Application Design
* Insecure Data Handling
* Injection (SQL, XSS)
* Security Misconfigurations





\## Goals

* Strengthen application security knowledge
* Understand common web vulnerabilities
* Practice exploitation and remediation techniques
* Build hands-on experience for DevSecOps roles





\## Tools used



* TryHackMe
* OWASP ZAP
* Burp Suite
